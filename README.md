# きょむじゃんけん
 
「#本田とじゃんけん」をモチーフとしているじゃんけんアプリです。

このゲームでは虚無感を体感できます！
 
## DEMO
 
準備中です
 
## Features

- 勝率をかなり低めに設定

- 勝ち負けに応じてランダムなコメントを表示

   - 勝ち負けが分かりやすいようにコメント表示部分の背景色を区別
   - さらに特定の連敗回数に応じて特別コメントあり
 
- 勝負回数、連敗回数の記録&表示

- メイン画面に戻ることで記録をリセット
